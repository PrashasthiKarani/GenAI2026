# GenAI2026: Agent Customization Guide

## Project Overview
This is a **beginner-friendly guide to Python data science basics** for learning Generative AI concepts. The project uses Jupyter notebooks to teach practical examples of numpy and pandas operations.

### Project Structure
- **Basics_numpy/**: NumPy array operations and fundamentals
- **Basics_Pandas/**: Pandas DataFrame operations and data manipulation
- **Basics_DataTypes/**: (placeholder for data type tutorials)
- **Basic_Operations/**: (placeholder for basic operation tutorials)
- **readme.md**: Project documentation with pandas examples

## Key Conventions

### Jupyter Notebooks
- Store notebooks with `.ipynb` extension in their respective `Basics_*` folders
- Each notebook should demonstrate practical, hands-on examples
- Include markdown cells to explain concepts and code cells with runnable examples
- Always test notebooks by running cells before finalizing

### Content Guidelines
- Keep examples beginner-friendly with clear variable names
- Add comments explaining what each operation does
- Include both simple examples and progressively complex patterns
- Reference relevant pandas/numpy documentation in comments

### Libraries
- **NumPy**: Array operations, mathematical functions, multi-dimensional arrays
- **Pandas**: DataFrames, Series, data selection, filtering, transformations, aggregations

## Agent Tasks
When working on this project, focus on:
1. **Expanding notebooks**: Add new cells demonstrating additional operations or edge cases
2. **Creating new modules**: Add notebooks to empty folders (Basics_DataTypes, Basic_Operations)
3. **Improving documentation**: Update readme.md with new examples and explanations
4. **Running and testing**: Execute notebook cells to validate correctness before completion

## Important Notes
- Notebooks should be self-contained learning materials, not production code
- Target audience is beginners with basic Python knowledge
- Examples should be practical and immediately understandable
